About

The entirely open source and free browser extension "Historia" functions as a meta link collector and searcher for websites all over the world.

It has the following features and advantages:

  • For Firefox, Opera, Microsoft Edge and Chrome based browsers + online version for all users.
  • Currently around 370 links with descriptions integrated.
  • The collected websites are in the following languages: English, German, French, Italian, Spanish, Catalan, Hungarian, Finnish, Japanese, simplified Chinese and traditional Chinese, Korean, Russian.
  • Time span: Antiquity - (mainly Early) Modern Period
  • The links are tested from time to time to avoid dead links.
  • For students, lecturers and amateurs to find already known or new websites.
  • The extension is written in / with: HTML, Javascript, jQuery 3.6.0, jQuery DataTables 1.12.1, CSS, JSON, manifest 2V for Firefox, manifest 3V for Microsoft Edge, Opera and Chrome.


DISCLAIMER: Please note that the extension, developer and contributors are not responsible for the content of the links. The extension "Historia" is a voluntarily work to help researchers all around the world.

How to download the extension

Firefox

#option1: download and activate with Firefox Add-Ons





#option2: download the source code from Github and install it manually.





Opera Beta / Opera Developer

#option1: download and activate with Opera Addons

This is only working with Opera BETA and Opera DEVELOPER! The normal Opera version is currently in the process of getting approved by Opera.





#option2: download the source code from Github and install it manually.

How to send in suggestions

If you have any suggestions, want a website to be added or want to report a problem, you have the following options:

#option1: Program it yourself and make a pull-request on Github.

#option2: Please write an eMail to the main developer of the extension. For suggestions, please provide the link of the website, its official name and a short description of the website in English.
